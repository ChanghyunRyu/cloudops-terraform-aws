# Terraform 모듈

🇺🇸 [Click here](README.md) to view this document in English.

이 디렉토리는 `shared`, `dev`, `prod`와 같은 전체 인프라 환경을 구성하기 위해 조합 가능한 재사용 가능한 Terraform 모듈들을 포함하고 있습니다.

각 모듈은 독립적으로 구성되어 있으며, VPC, EKS, S3 등의 특정 인프라 컴포넌트에 집중하여 재사용성과 유지보수성을 높입니다.

모듈별 사용 방법과 변수 설명은 각 디렉토리 내의 `README.md` 파일에 포함되어 있습니다.

---

## 📁 디렉토리 구조

```
modules/
├── vpc/               # VPC 프로비저닝 모듈
├── eks/               # EKS 클러스터 모듈 (클러스터, 노드, IAM, 애드온 포함)
├── argocd/            # ArgoCD 모듈
```

---

## 📚 모듈 문서 구성 방식

각 모듈에는 고유한 `README.md`가 포함되어 있으며, 일반적으로 다음과 같은 구조를 따릅니다:

| 섹션           | 설명                                                                 |
|----------------|----------------------------------------------------------------------|
| 헤더           | 모듈 이름, 선택적 언어 전환 링크 (예: 🇰🇷 한글 버전 링크)            |
| 개요           | 해당 모듈이 수행하는 역할 및 포함하는 인프라 구성 요소 설명          |
| 사용법         | Terraform 예제 코드와 함께 입력값 설정 방법 안내                    |
| 변수 설명      | 변수명, 타입, 필수 여부, 설명 등을 담은 표                          |
| 하위 모듈      | (해당되는 경우) 내부 구조나 중첩 모듈 구성에 대한 설명              |
| 참고 사항      | 주의사항, 팁, provider 관련 이슈 등 특별히 고려할 사항들            |

이러한 구성 방식을 통해 모든 모듈 문서를 일관되게 유지하며, 다양한 환경에서 손쉽게 모듈을 이해하고 적용할 수 있도록 돕습니다.
